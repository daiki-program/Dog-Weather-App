# Dog-Weather-App
OpenWeatherMap APIを活用した、愛犬家向け天気予報Webアプリ。Flaskでのバックエンド構築に加え、API制限を考慮した自作キャッシュ機構を実装。

# 🐕 ワンコ天気予報 (Dog Weather App)

天候に合わせてワンコがアドバイスをくれる、癒やし系天気予報アプリです。

### 🌟 技術的なこだわりポイント
- **レスポンスの高速化**: `datetime`モジュールを活用した1時間のキャッシュ機能を自作。不要なAPI通信をカットし、高速表示を実現しました。
- **セキュリティ**: APIキーは環境変数（`.env`）で管理し、GitHub公開時の安全性を確保。
- **UX**: OpenWeatherMapのアイコンとDog APIの画像を組み合わせ、視覚的に楽しいUIを目指しました。
